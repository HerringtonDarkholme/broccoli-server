# broccoli-plugin-base
Base classes for Broccoli plugins

## Development
* Create a new virtualenv
    * For example, using `pipenv`
    ```bash
    pipenv shell
    rm Pipfile Pipfile.lock
    ```
* Install dependecies under the virtualenv
    * For example, using `pipenv`
    ```bash
    pipenv run setup.py develop
    ```
